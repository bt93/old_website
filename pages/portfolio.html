Portfolio

Arcade Game Project

A recreation of the video game "Frogger" for Udacity's Front-End Web Development Nanodegree. Try to reach the water using the arrow keys on your keyboard without colliding with the enemy bugs. If you reach the end, you get a point. If you collide with a bug, you lose a life. If you get to zero lives, you go back to zero points. This game was used to demonstrate knowledge in object-oriented JavaScript.

View the github repo.

Code Your Own Quiz Project

This was a simple game created with Python. Upon opening the program you are given a choice between three levels of difficulty. When you type in a level you'll be asked to give a certain number of tries you'd like and then you'll try to type in the correct answer.
